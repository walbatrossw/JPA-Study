# 06. 다양한 연관관계 매핑

## 1) 다대일

- N:1 단방향
    - `Member.java`
        ```java
        @Entity
        public class Member {
        
            @Id
            @GeneratedValue
            @Column(name = "MEMBER_ID")
            private Long id;
        
            private String username;
        
            @ManyToOne
            @JoinColumn(name = "TEAM_ID")
            private Team team;
        
            // GETTER, SETTER ...
        }
        ```
    - `Team.java`
        ```java
        @Entity
        public class Team {
            @Id
            @GeneratedValue
            @Column(name = "TEAM_ID")
            private Long id;
        
            private String name;
        }
        ```
